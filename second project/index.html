HTML: HyperText Markup Language
HTML describes the structure of a web page,it consists of a series of elements that tell the browser how to display the content. HTML elements label pieces of content such as "its a heading", "its a pragraph" etc., 
History: 1960---USA---ARPANET
         1991---HTML---DHTML---XML---XHTML
         1993---CSS
HTML proceeds tag based approach,means everything inside HTML must follow with a tag

  • The DOCTYPE html declaration defines that this document is an HTML5 document
  • The html element is the root element of an HTML page, it wrapes the entire html document
  • The head element contains meta information about the HTML page
  • The title element specifies a title for the HTML page (which is shown in the browser's title bar pr in the page's tab)
  • The <Body element defines the document's body, and is a container for all the visible contents,such as headings, paragraphs,anchor tags images, hyperlinks, tables, lists etc.,
  • The H1 element defines a large heading
  • The p element defines a paragraphs

    • Structure of HTML

  • it begins with HTML opening tag and close with HTML closing tag
    • i.e.[tagname] data[/tagname] where (tagname=html)

    Self-closing tags (or) empty tags

      <br> : The <br> element is used for a line break in a text (also known as carriage-return) used where the long sentence needs breaks for clear visibility.

      <hr> : A thematic break between paragraph-level components represented by the <hr> element. E.g. change of shift in the topic/section.

      <img> :The <img> element is used to embed an image into the document.

      There are several other empty tags like <link> , <meta> , <input> , <embed> etc.,
