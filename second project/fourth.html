Git & Github

Git :
It is an application that is primarily concerned with version control where all the previous and updated version are stored.
Github :
It is a platform for developers in which we can deploy,store and merge the code

How to push code into Github

Prerequisite: To push the code first install git and github into your system then connect your github account to vscode by signing up

In Vscode open Terminal---> create new terminal--->then do the following commands.Type
  1. git init---> it initiates Git into your current directory
  2. git add (specific filename)/git add .---> It adds all the files in your directory
  3. git commit -m "message"---> It commits the current directory to git,once this is done your code is finalised.So before committing the directory you should have finalised the code to be pushed.
  4. when u first commit to Git it throws following errors:

    Git config errors :
      git config --global user.email "Your email id"
      git config --global user.name "username"
    then git add . and git commit
  5. git remote add origin "link"(here link specifies to your github repsitory url)
  6. git push -u origin main/master

The code is pushed now,check by Opening Github url
